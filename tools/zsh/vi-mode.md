# Zsh Vi Mode

## 启用方式

在 `~/.zshrc` 中添加：

```shell
bindkey -v
```

或使用 oh-my-zsh 插件 `vi-mode`（功能更丰富）：

```shell
plugins=(
    git
    vi-mode
    zsh-autosuggestions
    zsh-syntax-highlighting
)
```

> **插件额外功能**：`vv` 可在 $EDITOR 中编辑当前命令行。

## 模式说明

| 模式        | 说明                           | 进入方式           |
| ----------- | ------------------------------ | ------------------ |
| **Normal**  | 默认模式，用于移动和操作       | `Esc` / `Ctrl + [` |
| **Insert**  | 输入文本                       | `i` `a` `I` `A` 等 |
| **Visual**  | 选择文本（需 oh-my-zsh 插件）  | `v`                |
| **Replace** | 替换模式                       | `R`                |

## Normal 模式 — 光标移动

### 基础移动

| 按键       | 功能                          |
| ---------- | ----------------------------- |
| `h`        | 左移一个字符                  |
| `l`        | 右移一个字符                  |
| `j`        | 下一行（多行编辑时）          |
| `k`        | 上一行（多行编辑时）          |
| `w`        | 跳到下一个词首                |
| `b`        | 跳到上一个词首                |
| `e`        | 跳到当前/下一个词尾           |
| `0`        | 移动到行首                    |
| `^`        | 移动到行首第一个非空字符      |
| `$`        | 移动到行尾                    |

### 单词移动补充

| 按键       | 功能                                  |
| ---------- | ------------------------------------- |
| `W`        | 跳到下一个空格分隔的词首              |
| `B`        | 跳到上一个空格分隔的词首              |
| `E`        | 跳到当前/下一个空格分隔的词尾         |

### 字符查找

| 按键         | 功能                           |
| ------------ | ------------------------------ |
| `f{char}`    | 向右跳到字符 {char}            |
| `F{char}`    | 向左跳到字符 {char}            |
| `t{char}`    | 向右跳到 {char} 前一个位置     |
| `T{char}`    | 向左跳到 {char} 后一个位置     |
| `;`          | 重复上次 f/F/t/T               |
| `,`          | 反向重复上次 f/F/t/T           |

## Normal 模式 — 编辑操作

### 删除

| 按键       | 功能                           |
| ---------- | ------------------------------ |
| `x`        | 删除光标处字符                 |
| `X`        | 删除光标前字符                 |
| `dh`       | 删除光标前一个字符             |
| `dl`       | 删除光标处字符（等同 `x`）     |
| `dw`       | 删除到下一个词首               |
| `db`       | 删除到上一个词首               |
| `d$` / `D` | 删除到行尾                     |
| `d0`       | 删除到行首                     |
| `dd`       | 删除整行（剪切）               |

### 修改

| 按键         | 功能                              |
| ------------ | --------------------------------- |
| `cw`         | 删除到词尾并进入 Insert 模式      |
| `cb`         | 删除到词首并进入 Insert 模式      |
| `c$` / `C`   | 删除到行尾并进入 Insert 模式      |
| `c0`         | 删除到行首并进入 Insert 模式      |
| `cc` / `S`   | 删除整行并进入 Insert 模式        |
| `s`          | 删除当前字符并进入 Insert 模式    |

### 替换

| 按键         | 功能                          |
| ------------ | ----------------------------- |
| `r{char}`    | 替换当前字符为 {char}         |
| `R`          | 进入 Replace 模式（逐字覆盖） |

### 大小写

| 按键       | 功能                     |
| ---------- | ------------------------ |
| `~`        | 切换当前字符大小写并右移 |

### 复制粘贴

| 按键       | 功能                           |
| ---------- | ------------------------------ |
| `yw`       | 复制一个词                     |
| `y$` / `Y` | 复制到行尾                     |
| `y0`       | 复制到行首                     |
| `yy`       | 复制整行                       |
| `p`        | 在光标后粘贴                   |
| `P`        | 在光标前粘贴                   |

### 撤销 & 重复

| 按键       | 功能                          |
| ---------- | ----------------------------- |
| `u`        | 撤销                          |
| `Ctrl + R` | 重做（需 oh-my-zsh 插件）     |
| `.`        | 重复上次编辑操作              |

## Insert 模式 — 进入方式

| 按键       | 功能                                  |
| ---------- | ------------------------------------- |
| `i`        | 在光标前插入                          |
| `I`        | 在行首插入                            |
| `a`        | 在光标后插入                          |
| `A`        | 在行尾插入                            |
| `o`        | 在下方新行插入（多行编辑时）          |
| `O`        | 在上方新行插入（多行编辑时）          |
| `s`        | 删除当前字符并插入                    |
| `S`        | 删除整行并插入                        |

## Insert 模式 — 快捷键

| 按键              | 功能                       |
| ----------------- | -------------------------- |
| `Esc` / `Ctrl + [` | 回到 Normal 模式          |
| `Ctrl + H`        | 删除前一个字符（等同 ⌫）   |
| `Ctrl + W`        | 删除前一个词               |
| `Ctrl + U`        | 删除到行首                 |
| `Ctrl + M` / `Enter` | 执行命令               |

## 历史操作

| 按键              | 功能                         |
| ----------------- | ---------------------------- |
| `k` / `↑`         | 上一条历史                   |
| `j` / `↓`         | 下一条历史                   |
| `/pattern`        | 反向搜索历史（输入后按 Enter）|
| `n`               | 继续搜索下一个匹配           |
| `N`               | 反向搜索上一个匹配           |
| `G`               | 跳到历史末尾（最新）         |
| `gg`              | 跳到历史开头（最早）         |
| `{n}G`            | 跳到第 n 条历史              |

## Visual 模式（oh-my-zsh 插件）

| 按键       | 功能                           |
| ---------- | ------------------------------ |
| `v`        | 进入 Visual 模式               |
| `h` / `l`  | 扩展选区                      |
| `w` / `b`  | 按词扩展选区                  |
| `y`        | 复制选中内容                  |
| `d`        | 删除选中内容                  |
| `c`        | 删除选中并进入 Insert 模式    |
| `Esc`      | 退出 Visual 模式              |

## 重复 & 计数

所有移动和操作都支持数字前缀重复：

```shell
3w      # 向右跳 3 个词
5dh     # 向左删除 5 个字符
2dw     # 删除 2 个词
d3w     # 同上，删除 3 个词
```

## 常用组合速查

| 场景                   | 按键                |
| ---------------------- | ------------------- |
| 删除整行重新输入       | `cc` / `S`          |
| 修改一个词             | `cw`                |
| 删除到行尾             | `D` / `d$`          |
| 在行尾追加内容         | `A`                 |
| 替换一个字符           | `r{char}`           |
| 搜索历史中的 git 命令  | `/git` Enter        |
| 重复上次编辑           | `.`                 |
| 撤销                   | `u`                 |
| 快速跳到某个字符       | `f{char}` / `t{char}` |

## 参考配置

```shell
# ~/.zshrc 推荐配置

# 启用 vi 模式
bindkey -v

# 或使用 oh-my-zsh 插件
# plugins=(git vi-mode)

# vi-mode 插件: 设置 jk 快速回到 Normal 模式
# bindkey -M viins 'jk' vi-cmd-mode

# 减少 Esc 延迟（默认 0.4s 太长）
export KEYTIMEOUT=1

# 光标样式区分模式（需终端支持）
# Insert 模式: 竖线 |   Normal 模式: 方块 ▏
```

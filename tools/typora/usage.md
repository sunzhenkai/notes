# 插入图片

![image-20210818230821747](usage/image-20210818230821747.png)

- 选择「复制到指定路径」+ 「`./${filename}`」
- 勾选「优先使用相对路径」

这样，不管是复制文件还是从剪切板复制图片，都会保存到文件同目录、同文件名下，引用的时候使用相对路径，这样，在上传到 git 仓库时，可以同步上传图片，并保持引用关系。

# 说明

Poetry 用于管理 Python 项目信息及依赖。

# 安装

```shell
curl -sSL https://install.python-poetry.org | python3 -
```

# Workflow

## 初始化

```shell
poetry init
```

# Run

```shell
poetry run python ...
```

# Activate

```shell
poetry env activate
```

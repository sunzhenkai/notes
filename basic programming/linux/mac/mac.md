# 问题

## 拷贝文件导致图标变灰无法访问

尝试 [这里](https://blog.csdn.net/evandeng2009/article/details/53242196) 无果。

```shell
# 仅适用文件夹
# 命令行输入如下内容
exitFun() {
  echo "ERROR: failed, due to $1"
  exit 1
}

fix() {
    [ ! -d "${1}" ] && exitFun "target is not folder"
    mv "${1}" "${1}_back"
    mkdir "${1}"
    mv "${1}_back"/* "${1}/"
    mv "${1}_back"/.[^.]* "${1}/"
    rmdir "${1}_back"
}

# 修复，输入 fix <文件夹目录> OR 输入 fix [拖拽文件夹至终端]
$ fix <path>	
```

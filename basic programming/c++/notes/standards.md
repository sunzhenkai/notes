# 版本说明

WG21（The ISO C++ committee）有严格的时间限制，每 3 年推出一版新的标准。最新的版本信息参考 WG21 [官网](https://isocpp.org/)。每个正式标准确定前，会使用草案名（draft），比如 `c++1z`、`c++2a`。

# 初衷

- 想组装一台可以随便折腾的主机
- 性价比要高，不要太贵
- 考虑到后面可能要部署大数据套件，性能不能太弱

# 注意事项

- 主板两个 cpu 供电口距离过大，电源需要两条独立 cpu 供电线，如果买一拖二的 cpu 供电线电源，提前买一根 cpu 供电线
- 注意散热器支持的接口，我这里用的是兼容 2011 接口的

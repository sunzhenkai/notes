# 网络问题

- 找到一个可以访问的 IP

  - [站长工具](https://ping.chinaz.com/github.com)
  - [site24x7](https://www.site24x7.com/ping-test.html)

- 修改 hosts

  - `sudo vim /etc/hosts`
  - 添加记录

  ```shell
  <ip>	github.com
  ```

# 听点什么

无聊的时候，或者什么都不想做、但又不想太安静，想听点什么的时候，可以试下下面的几个音频栏目。

- 《观棋有语》，蜻蜓 FM 或 B 站，主要聊些国内国外的时事
- 《察话会》，蜻蜓 FM，主要聊些军事相关的事情

# 看点什么

发呆不知道看些什么，又不想从头刷剧，可以考虑下下面的视频。

- 《武林外传》
- 《地下交通站》
- 《乡村爱情》系列
- 《家有儿女》

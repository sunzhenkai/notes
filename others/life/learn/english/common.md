# 地址

**格式**

| 级别 | 英文    | 示例          |
| ---- | ------- | ------------- |
| 室   | Room    | Room 602      |
| 村   | Vallage | Huaxi Vallage |
| 号   | No.*    | No.12         |
|      |         |               |



```
Ｘ室 Room Ｘ
X 村 *** Vallage,如华西村 Huaxi Vallage.
Ｘ号 No. X
Ｘ单元 Unit Ｘ
Ｘ号楼 Building No. Ｘ
Ｘ街 Ｘ Street
Ｘ路 Ｘ Road
Ｘ区 Ｘ District
Ｘ县 Ｘ County
Ｘ镇 Ｘ Town
Ｘ市 Ｘ City
Ｘ省 Ｘ Province
```

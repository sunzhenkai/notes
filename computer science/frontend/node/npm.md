# install pkg

```shell
$ npm install <pkg-name> # OR npm install <pkg-name>
$ npm install <pkg-name>@<version> -s
```

# Upgrade pkg

```shell
$ npm update <pkg-name> --save # OR --save-dev
```

**使用工具更新所有依赖**

```shell
$ npm i -g npm-check-updates # 安装依赖
$ ncu -u                     # 升级
$ npm install								 # 安装升级后的依赖
```

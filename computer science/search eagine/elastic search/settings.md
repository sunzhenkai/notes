# 修改配置

## 接口修改

```shell
# dev tools
PUT <index>
{
    "settings" : {
        "index" : {
            "<name>" : <value>
        }
    }
}

# curl
curl -X PUT -u <user>:<password> "http://<host>:<port>/<index>" -d
```

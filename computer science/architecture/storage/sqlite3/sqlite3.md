# 打开 db 文件

```shell
sqlite3 db.sqlite3
```

# Query

参考 [sqlite.org/cli](https://www.sqlite.org/cli.html)。

## 查看表

```shell
.tables
# OR
SELECT name FROM sqlite_master WHERE type='table';
```

## 查看表创建语句

```shell
.schema <tablename>   # 去掉 <> , 不用 ; 结尾
```

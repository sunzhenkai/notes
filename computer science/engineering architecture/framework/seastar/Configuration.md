# 系统配置

## aio-max-nr

```shell
$ sudo vim /etc/sysctl.conf
# 添加
fs.aio-max-nr = 1048576
$ sudo sysctl -p /etc/sysctl.conf
```

## perf_event_open

```shell
$ sudo vim /etc/sysctl.conf
# 添加
kernel.perf_event_paranoid = -1
$ sudo sysctl -p /etc/sysctl.conf
```

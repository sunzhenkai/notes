# 安装

可以从 [清华源](https://mirrors.tuna.tsinghua.edu.cn/grafana/) 下载。

```shell
# centos
wget https://mirrors.tuna.tsinghua.edu.cn/grafana/yum/rpm/grafana-7.5.3-1.x86_64.rpm
sudo npm install grafana-7.5.3-1.x86_64.rpm
```

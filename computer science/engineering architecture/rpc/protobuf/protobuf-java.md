# 对比

## StringValue & string

类似的还有 `int32 & Int32Value` 等。

`StringValue` 和 `string` 相比：

- StringValue 可以为 null，在部分语言中，string 类型数据不能为 null，那么对于 string 类型，无法区分 `""` 是代表 null 还是空字符串
- gRPC 调用过程的的参数和返回值不能为基本类型，也就是不能为 string、int32 这种
